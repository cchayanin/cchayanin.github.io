Position
position z-index
เป็น property สำหรับระบุถึงวิธีการแสดงตำแหน่งของ element
  • static เป็นค่าเริ่มต้น แสดงผลตามลำดับ document flow
  • absolute แสดงผลตามค่าที่กำหนด โดยอย่างอิสระไม่สน element ก่อนหน้า
  • fixed แสดงผลอ้างอิงตามตำแหน่งหน้าจอ
  • relative แสดงผลคล้าย static แต่กำหนดตำแหน่งด้วย top left bottom right ได้
  • sticky มีความสามารถของ releative กับ fixed มักใช้กับหัวข้อที่เรื่องรายละเอียดเยอะเพื่อยังคงแสดงหัวข้อเมื่อ scroll mouse ลงมา
เป็น property เพื่อกำหนดลำดับการซ้อนทับของ element ใช้เมื่อ element มีการซ้อนทับกันแล้วต้องการเปลี่ยนการซ้อนทับ
