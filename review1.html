CSS syntax ประกอบด้วยสองส่วนคือ Select และ Declaration Block ซึ่งมีได้ตั้งแต่หนึ่ง Declaration Block ขึ้นไปคั่นแยกจากกันด้วย semicolon(;) ในแต่ละ Declation Block จะประกอบด้วยได้ค่า Property และ Value การเขียน CSS มีด้วยกันสามแบบ คือ
  1. Inline - เขียนที่ html element ตัวนั้น ด้วย attribute style
  2. Internal - เขียนในไฟล์ html ด้วย <style> โดยภายใต้ <head>
  3. External - เขียนแยกเป็นไฟล์ CSS ต่างหาก

CSS Syntax Example

h1 {color:blue; font-size:12px}

CSS Writing Example

  • Inline : 
                <h1 style="color:blue;text-align:center;">This is a heading</h1>
  • Internal : 
                <head>
                <style>
                    body {
                    background-color: linen;
                    }

                    h1 {
                    color: maroon;
                    margin-left: 40px;
                    }
                </style>
            </head>
  • External : 
                <head>
                <link rel="stylesheet" type="text/css" href="mystyle.css">
            </head>
